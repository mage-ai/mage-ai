## Resizable window

Adjust the width and/or height of the application window.

<Frame>
    <p align="center">
    <img
        alt="Resizable window"
        src="https://github.com/mage-ai/assets/raw/main/dev-ux/applications/resizable.gif?raw=true"
    />
    </p>
</Frame>

1. Drag the horizontal (left, right) edges to resize the width. 
1. Drag the bottom edge to resize the height.
1. Drag any corner to resize the width and height simultaneously.

---
