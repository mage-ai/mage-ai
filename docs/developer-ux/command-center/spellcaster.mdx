The Command Center is an ancient magical artifact that imbues its wielder with godlike
capabilities such as:

1. <b>Teleport</b> between different pages (*go to different pages*).
1. <b>Cast</b> spells that trigger pipelines, immortalize code, etc. (*perform an action*).
1. <b>Conjure</b> new entities such as files, folders, pipelines, and more (*create new objects*).
1. <b>Enchant</b> objects and reveal their inner details (*view object details*).
1. <b>Summon</b> applications to do your bidding and fight alongside you (*open applications*).
1. <b>Heal</b> wounds suffered from bugs and errors (*get help and support*).
1. <b>Alchemize</b> entities and update their inherent properties (*update object attributes*).

---

## Setup

<Note>
    Available on versions `>= 0.9.60`
</Note>

To enable this feature:

<Steps>
  <Step title="Project settings">
    Go to your [project settings](http://localhost:6789/settings/workspace/preferences).
  </Step>
  <Step title="Enable feature">
    Enable the feature named <b>Command Center</b>.
  </Step>
  <Step title="Optional">
    Restart Mage so that your files can be re-indexed.
  </Step>
</Steps>

## Launch Command Center

There are several ways to launch the Command Center:

<AccordionGroup>
    <Accordion title="Header dropdown menu" icon="bars" iconType="duotone" defaultOpen>
        1. In the top right corner of the screen, click on your avatar.
        1. In the dropdown menu, click the option labeled <b>Launch Command Center</b>.

            <p align="center">

    <Accordion title="Keyboard shortcuts" icon="keyboard" iconType="duotone" defaultOpen>

## Modes

The Command Center by default will show a standard list of actions and items that are
useful throughout Mage.

However, there comes a time when you must transform into a different mode to better tackle
specific quests. Currently supported modes are:

<Card title="Version Control" icon="code-branch" iconType="duotone" color="#FFCC19">
    Switch into the Version Control mode to become the master of Git workflows.
    This mode comes with spells and artifacts for
    pulling code, viewing file diffs, committing changes, pushing features, etc.
</Card>

*More modes coming very soon!*

## Application launcher

The Command Center can launch Mage applications within the currently running Mage instance. 
These applications can be used simultaneously throughout your journey in Mage.

Currently supported applications:

<Card title="Arcane Library" icon="scroll" iconType="duotone" color="#7D55EC">
    This is the file browser and the code editor self-contained in a single application.
</Card>

<Card title="Portal Terminal" icon="person-to-portal" iconType="duotone" color="#FF1E59">
    This is the terminal with multi-terminal support and can be launched and used on any page.
</Card>

<Card title="Code Matrix" icon="code-simple" iconType="duotone" color="#8ADE00">
    This application provides a new and revolutionary way to write code, test, and debug.
</Card>

*More applications coming very soon!*
