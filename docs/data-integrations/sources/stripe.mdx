## Configuration

To set up the Stripe source, provide the following configuration parameters:

| Key              | Description                                                   | Sample Value        | Required |
| ---------------- | ------------------------------------------------------------- | ------------------- | -------- |
| `account_id`     | Your Stripe account’s unique ID.                               | `acct_73bb...`      | ✅        |
| `client_secret`  | Secret key associated with your Stripe account.               | `sk_live_ABC1...`   | ✅        |
| `start_date`     | The date to start syncing data from. Format: `YYYY-MM-DD`.     | `2022-01-01`        | ✅        |

> **Find your API keys:**  
> Refer to Stripe’s [API keys documentation](https://stripe.com/docs/keys) to locate your `client_secret` and `account_id`.
