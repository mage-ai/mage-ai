## Style guide

Mage follows the [Airbnb JavaScript style guide](https://airbnb.io/javascript/react/).

### Linter

To run the linter locally, execute this script:

```bash
./scripts/test.sh
```

## Setup

```bash
yarn global add eslint-config-next
yarn global add eslint_d
```

Looking for a curated list of open issues to use as frontend engineer interview tasks? Check out the [Frontend Interview Tasks](/contributing/frontend/interview-tasks) guide for issues organized by difficulty, with skill breakdowns and a suggested evaluation rubric.
