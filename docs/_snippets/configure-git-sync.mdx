_If_ using Git Sync, tic the box labeled "use One-way Git Sync". You can read more about Git Sync [here](/production/data-sync/git-sync). If One-way Git Sync is enabled, you'll also need to provide:
- `Branch Name`: The name of the branch you want to sync with.
- `Include submodules`: If your repo has submodules, tic this box to include them in the sync.
- `Sync before each trigger run`: If you'd like a sync to occur before a trigger run to ensure that the latest code is used, tic this box.
- `Sync on server start up`: If you'd like to sync on server start up, tic this box.

Otherwise, continue to the next step.
